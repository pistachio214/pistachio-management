# Pistachio Manager

与 `Pistachio` 配套的通用管理系统

## 服务端支持

[https://github.com/pistachio214/pistachio-api](https://github.com/pistachio214/pistachio-api)

## 技术方案

`React18` + `Antd5` + `craco`

追求用最简洁的技术方案完成此管理端

## 涉及模块

`管理员模块`、`角色模块`、`菜单模块`、`字典模块`、`日志模块`
